# FlaskBook — Flask Practice Project (CS50)

Practice app built during CS50 to consolidate Flask fundamentals.

## Features
- User registration and login by ID
- Session management with flask_session
- Add/remove sports per user
- Real-time AJAX search
- SQLite database with 3 tables and foreign keys

## Stack
Python · Flask · SQLite · Jinja2 · JavaScript · CSS

## Note
This is a CS50 practice exercise, not a production project.
